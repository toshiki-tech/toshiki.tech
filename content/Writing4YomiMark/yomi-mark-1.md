# 为了能让阅读日语片假名外来语更丝滑，做了一款Chrome插件 —— YomiMark

对于一个还在学习日语阶段的技术开发者来说来说，看日本的技术网页最痛苦的时刻莫过于：

* 满屏汉字（Kanji），认识意思却不知道怎么念；

* 满屏外来语（片假名Katakana），每个假名都认识，拼在一起却不知道是什么意思（“コンテキスト”明明是英语 Context，却要反应半天）。

💡 开发的初衷：为什么是 YomiMark？

在开发 YomiMark 的过程中，我保持三个核心原则：

1. 极简主义：划词注音，拒绝干扰

我不希望插件破坏网页原本的美感。YomiMark 采用了“划词触发”的逻辑——只有当你选中一段文字并点击紫色的小按钮时，注音才会优雅地出现在汉字上方。这种交互方式鼓励“先读再查”，对日语学习者的记忆更有帮助。

2. 不止是汉字：更是外来语的“救星”

这是 YomiMark 的一个亮点。我们不仅处理汉字注音，还内置了外来语词典。当你选中诸如“イノベーション”这类词汇时，它不只是标注假名，还能直接告诉你它的英文原词（Innovation）。这对于在技术文档中“冲浪”的初学者来说，简直是神器。

3. 隐私与速度：纯离线运行

我不相信所有东西都要“上云”。YomiMark 将形态素解析器（kuromoji.js）和完整的词典包直接封装在了插件里。这意味着你选中的内容永远不会发送到后端服务器。不仅保护了个人的阅读隐私，更保证了“指哪打哪”的瞬时响应，无需因网络延迟而等待。

🛠️ 强大功能一览

* 智能分词：基于强大的解词引擎，能够精准识别词汇边界。

* 原生渲染：采用 HTML 标准的 <ruby> 标签实现，视觉效果就像教科书般自然，完美适配各种字体风格。

* 深色模式：我们为插件 Popup 设计了极其简洁、高级的深黑主题，审美始终在线。

* 高度定制：觉得默认颜色不够醒目？你可以在设置里自由调整汉字和外来语注音的颜色。

🌍 使用场景：谁需要 YomiMark？

* 职场IT员工：在翻阅日本公司的官网或技术文档时，快速掌握专业词汇读音。

* 新闻爱好者：读 NHK News 或 Yahoo Japan 时，不再被复杂的地名和政治词汇挡住脚步。

* 日语备考者：通过真实网页阅读训练语感，随划随读。

🚀 如何开始？

目前 YomiMark 处于快速迭代的 V1 阶段，你可以在 Chrome 应用商店中https://chromewebstore.google.com/detail/ldfcjpnjokhdoihapfcnimchieaofhin?utm_source=item-share-cb 查看详情，下载体验。

1. 选中文本：像平常复制文字一样选中你想看的日语。

2. 点击“読”：文字上方会出现一个精致的小按钮。
   
3. 瞬间解锁：注音即刻呈现。
   
4. 个性化设置注音文字的颜色。
